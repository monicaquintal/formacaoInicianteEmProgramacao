GitHub

Serviço que fornece a possibilidade de criar repositórios Git!

O que é?

É uma alternativa para criarmos servidores remotos gratuitamente, compartilháveis pela internet!

É uma plataforma de hospedagem de código-fonte e arquivos com controle de versão usando o Git. Ele permite que programadores, utilitários ou qualquer usuário cadastrado na plataforma contribuam em projetos privados e/ou Open Source de qualquer lugar do mundo.

Criando um repositório

  1. Definir o criador do repositório (Owner) e o seu nome (Repository name);
  2. Criar uma descrição (Description);
  3. Configurá-lo como público ou privado;
  4. Seguir os comandos para envio dos arquivo commitados.
